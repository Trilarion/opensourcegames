# Breakout 71

- Home: https://breakout.lecaro.me/
- Inspiration: lbreakoutHD, nuclear throne
- State: mature, maintained but almost no new development
- Play: https://breakout.lecaro.me/
- Download:  https://renanlecaro.itch.io/breakout71
- Platform: Windows, Linux, macOS, Android, iOS, Web
- Keyword: arcade, roguelite, breakout, bricks
- Code repository: https://gitlab.com/lecarore/breakout71
- Code language:  Typescript
- Code license: GNU AGPLv3
- Code dependency: Built with parcel, tested with jest, admin level editor in react, no runtime dependency 
- Developer: Renan LE CARO

This is a very lightweight web game about breaking bricks, collecting coins and picking game changing upgrades
between levels. It comes with 300+ levels, 75 upgrades, and 14 supported languages. 
That's a lot of content in a 800kb package.

## Building

- Build system: parcel, gradlew

Just git clone, npm install, then run start.sh or build.sh
