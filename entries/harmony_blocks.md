# Harmony Blocks

- Home: https://github.com/C-Jeril/harmonyblocks, https://github.com/tg12/harmonyblocks
- Inspiration: Tetris
- State: beta
- Keyword: arcade, browser, non-violent, relaxing, single-player
- Code repository: https://github.com/C-Jeril/harmonyblocks.git, https://github.com/tg12/harmonyblocks.git (@add)
- Code language: TypeScript, JavaScript, Python
- Code license: MIT

“falling-block” puzzle designed to promote focus and relaxation. Shapes drift downward and the player gently aligns them with faint outlines at the bottom of the screen.  
Soft gradients, blues & greens, adaptive speed and purely positive feedback.

## Building

Typescript:
npm install
npm run dev        # local server at http://localhost:3000
